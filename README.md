# Heart Disease Prediction App

A machine learning web application that predicts the likelihood of heart disease based on patient health data. Built using Scikit-learn and deployed with Streamlit to provide real-time predictions through an interactive interface.

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit  

## Features
- Predicts heart disease risk instantly  
- Interactive and user-friendly interface  
- End-to-end ML workflow from training to deployment  

## How to Run
```bash
streamlit run app.py
```
